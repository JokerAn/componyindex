产品优势

优势点

托管云

易用

资源申请秒级响应，极大缩短了业务应用上线的周期；云主机可以灵活挂载卸载云硬盘，添加删除网卡，绑定解绑公网IP，弹性调整配置，灵活便捷；通过VNC可视化界面，无需网络连通，也可以快速登陆云主机进行管理。

可靠

提供与龙腾云公有云相同的SLA服务保障，无论是底层架构，还是专业的运维团队，保证您的云时刻在线。持续不断产品更新，优化云平台性能，上线新的产品功能，你的云平台将在技术和功能保持领先。

安全

提供了全方位的网络和系统安全保 护,并通过稳固的底层构架保障用户数据的安全性和业务稳定性，精确到分钟级的实时云主机监控，支持自定义告警功能，保证您的业务稳定运行，支持VLAN\VXLAN网络隔离、软防火墙、数据灾备方案、平台各组件高可用。

节省成本

集中管理企业内部IT资源，通过云化，形成一个大的资源池，在有资源需求的时候进行统一调度，简化了IT基础设施的部署和管理，通过资源的云化，提升了资源的利用率、降低了整体TCO。

计算
托管云提供一种可随时自助获取、可弹性伸缩的云服务器，帮助用户打造可靠、安全、灵活、高效的应用环境，确保服务持久稳定运行，提升运维效率。

  • 秒级获取

    秒级启动速度，无论同时创建多少台，都可以在几十秒内完成。

  • 高弹性

    弹性可扩展，随时调整CPU、内存。

  • 自有映像

    可在搭建好环境后捕获成自有映像，之后可基于此自由映像创建更多主机。

  • 网页登录

    不需要绑定公网IP就可在网页上登录主机。

  • 全面监控

    历史和实时监控主机CPU、内存、硬盘读写、网络流量等重要数据。

镜像
镜像（Image）是一个包含了软件及必要配置的机器模版。 作为基础软件，操作系统是必须的，您也可以根据自己的需求将任何应用软件 （比如，数据库、中间件等）放入镜像中。

  • 多种操作系统

    我们为您提供最新、最常用、最稳定的镜像。

  • 共享镜像

    可将自有镜像选择分享给您的子帐户。

  • 自带存储空间

    Linux 自带 20G 高性能硬盘；Windows 自带 50G 高性能硬盘。

存储
统一的CEPH分布式存储，降低云平台存储成本，颠覆原生Open— stack存储架构，减少组件间的数据传输，让您的云平台体验更加 顺畅。

  • 分布式架构

    领先的云存储软件架构，多副本机制，带来更加安全的数据保障，实现数据容错、纠删功能，集成存储界面，实现服务可视化管理

  • 智能数据模型

    基于业务的数据冷热分层

    动态加速存储功能

    为业务提供多种存储资源池

    提供性能与容量的自由选择SSD+HDD

网络
基于Neutron、OVS、VXLAN、NameSpace等技 术,UStack托管云将所有物理世界的网络设备都被 设计成了可操作的网络对象。云管理面板,就可以根据业务需要灵活快速地构建 复杂的网络环境,使网络管理更加智能。

公网 IP 地址
公网 IP 地址是在互联网上合法的静态 IP 地址。 在龙腾云系统中，公网 IP 地址与您的账户而非特定的资源（主机或路由器）关联，在您创建主机时我们会为您的主机随机分配一个公网IP，一个服务只有一个公网IP。

  • 软件定义网络SDN

    灵活的组网方式 图形化的网络拓扑

    界面化定制您的私有网络

产品架构

TelCloud通过领先的技术，打造基于OpenStack的强大云平台。

TelCloud云平台采用超融合架构，在标准的x86服务器和通用网络设备的基础上，通过弹性计算、统一存储和SDN网络等技术，构建了涵盖云主机、云硬盘和VPC网络在内的完整云计算基础设施，为用户提供全功能的云计算平台。通过强大的运维监控系统和完善的持续集成体系将为TelCloud云平台不断增加新的功能，为用户持续交付新的价值。
